# 1pc-training-site

Static landing page for **1% Community** at `training.1percent.community`.
Mobile-first HTML/CSS, no framework, no build step.

## Structure

- `index.html` — single page, bilingual (AR / EN)
- `assets/css/style.css` — mobile-first, desktop via `@media (min-width: 768px)`
- `assets/js/main.js` — language toggle (vanilla JS)
- `assets/images/` — hero and program card images
- `CNAME` — GitHub Pages custom domain

## Local preview

Open `index.html` directly in a browser. No build step.

## Deploy

Hosted on GitHub Pages. Pushing to `main` publishes automatically.
